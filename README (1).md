## 📸 Project Overview

A **premium, feature-rich To-Do List App** built with pure **HTML, CSS, and JavaScript**.  
This project goes far beyond the basic requirements — it includes advanced productivity features like a Pomodoro Timer, Drag & Drop, Dark Mode, Voice Input, Analytics, and more.

---

## ✨ Features

### ✅ Core Requirements (Completed)
| Feature | Status |
|---|---|
| Add tasks | ✅ Done |
| Remove tasks | ✅ Done |
| Mark tasks as completed | ✅ Done |
| Store data using localStorage | ✅ Done |

### 🔥 Bonus Features
| Feature | Description |
|---|---|
| 🌙 Dark Mode | Light/Dark toggle, saved in localStorage |
| 🍅 Pomodoro Timer | 25/5/15 minute focus & break modes |
| 🏷️ Categories | Work, Study, Personal, Health |
| ✅ Subtasks | Add subtasks inside each task |
| 📝 Notes | Add description/note to any task |
| 🎤 Voice Input | Speak to add tasks (HTTPS) or type fallback |
| 🔔 Notifications | Browser push notifications for reminders |
| 🎉 Confetti | Celebration when all tasks complete |
| 🔥 Streak Counter | Daily productivity streak tracker |
| 📈 Weekly Analytics | Bar chart showing weekly completed tasks |
| 📊 Progress Circle | Animated SVG completion percentage |
| ⭐ Drag & Drop | Reorder tasks by dragging (SortableJS) |
| ✏️ Edit Modal | Edit task, notes, date, and subtasks |
| 🔃 Sorting | Sort by Newest, Oldest, Priority, Deadline |
| 🔍 Search | Real-time task search |
| 💡 Motivational Quotes | Random quotes for productivity |
| 📱 Responsive Design | Works on mobile & desktop |

## 🛠️ Technologies Used

- **HTML5** — Semantic structure
- **CSS3** — Glassmorphism design, CSS variables, animations
- **JavaScript (ES6+)** — All logic, localStorage, APIs
- **SortableJS** — Drag & Drop functionality
- **Canvas Confetti** — Celebration animation
- **Google Fonts** — Plus Jakarta Sans, DM Mono
- **Web APIs** — Notification API, SpeechRecognition API

## ⚙️ How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AwaisRamzan337/HexSoftwares_Todo_List_App.git
   ```

2. **Open the project:**
   ```bash
   cd HexSoftwares_Todo_List_App
   ```

3. **Run it:**
   - Simply open `index.html` in any browser
   - Or use VS Code **Live Server** extension for best experience

> ⚡ No installation, no dependencies, no build step required!
